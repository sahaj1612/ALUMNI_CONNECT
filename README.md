# Alumni Connect

Alumni Connect is a college-focused platform designed to connect students and alumni, encourage interaction, and support mentoring, communication, and campus engagement.

> Status: This project is currently under process and still being refined.

## Tech Stack

- PHP
- MongoDB PHP library
- HTML
- CSS
- JavaScript

## Current Scope

- Student and alumni login flows
- Homepage and panel pages
- Alumni and student interaction-oriented pages
- Asset and upload handling

## Project Structure

- `pages/` contains the main rendered pages.
- `api/` contains backend request handlers.
- `assets/` stores frontend styles and scripts.
- `config/` contains helper and configuration logic.
- `uploads/` stores uploaded content.

## Run Locally

Set up a local PHP server and make sure the required dependencies from `composer.json` are installed.

```bash
composer install
php -S localhost:8000
```

Then open the app in your browser from the served local address.

## Note

This repository is still in development, so some pages and flows may be incomplete or under active changes.
